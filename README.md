### 2. 기능 소개
- 나의 주식
  - 내가 갖고있는 국내 주식을 등록하여 실시간으로 주가, 수익률을 확인 할 수 있습니다.
- 수익 노트
  - 주식 매매를 통해 얻은 수익을 기록합니다.
- 메모
  - 주식관련 메모를 저장 할 수 있습니다.
- 마이
  - 내 정보 및 편의 기능 on/off, 로그아웃, 회원탈퇴를 할 수 있습니다. 

### 3. 기술 스택 변경 사항
 - SQLite -> Room
 - MVP -> MVVM
 - Koin -> Hilt
 - LiveData -> StateFlow
 - Gson -> Kotlinx Serialization
