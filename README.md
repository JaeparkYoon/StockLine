### 2. 기능 소개
- 수익 노트
  - 주식 매매를 통해 얻은 수익을 기록합니다.
- 메모
  - 주식관련 메모를 저장 할 수 있습니다.
- 마이
  - 내 정보 및 편의 기능 on/off, 로그아웃, 회원탈퇴를 할 수 있습니다. 

### 3. 추후 업데이트 계획
 - 현재 내가 가지고 있는 주식의 실시간 시세를 확인 할 수 있는 기능 추가 
 - 하단 배너 광고 추가
 - 현재 주가를 가지고 오는 기능 관련한 편의성 on off 추가

### 4. 기술 스택 변경 사항
 - SQLite -> Room
 - MVP -> MVVM
 - Koin -> Hilt
